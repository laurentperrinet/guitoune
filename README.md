# Guitoune: the mobile‑Friendly 6‑String Guitar Tuner

Why need an app when you can do it online ?



Note: En québécois, une *guitoune* est avant tout un synonyme familier et légèrement désuet de « cigarette », parfois détourné pour parler d’une petite guitare ou, de façon très informelle, d’un « machin » quelconque. À ne pas confondre avec une *guidoune*.
